# Phase 41 — Native Title Bar and Shell Order Fix

## الهدف
إصلاح ترتيب الشريط العلوي بعد Phase 40 بناءً على اختبار الواجهة الفعلي.

## التعديلات

1. استعادة شريط العنوان الأصلي للنظام:
   - إزالة `Qt.FramelessWindowHint` من `MainWindow`.
   - إزالة `Qt.WA_TranslucentBackground` من `MainWindow`.
   - إضافة `setWindowTitle("الراجحي للمحاسبة")`.
   - إبقاء شريط العنوان المخصص القديم مخفياً وغير مستخدم.

2. إعادة ترتيب الواجهة العلوية:
   - شريط القوائم/التنقل داخل `ModernTopBar` أصبح في الأعلى.
   - مربع البحث + الإشعارات + تبديل الثيم + المستخدم أصبحوا أسفل شريط القوائم/التنقل.

3. تبويب الإعدادات:
   - إضافة تنظيف يضمن وجود شريط إعدادات واحد فقط داخل الصفحة.
   - إزالة أي `ModernPageHeader` مكرر عند فتح صفحة الإعدادات.
