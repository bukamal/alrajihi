# Phase 455 — Targeted Screen Rebuild

Phase 455 is a narrow Windows-runtime visual rebuild for the remaining screenshot-visible legacy screens after Phase 454.

Scope:
- Login: compact custom chrome, form/brand panel hierarchy, field density.
- POS: barcode/table/payment-first surface, compact context bars, stronger payment footer.
- Invoice editor: header, quick entry, major grid, financial summary, sticky action footer.
- Material editor: card-based basic/pricing/barcode/units layout with persistent action footer.
- Dashboard: targeted panel roles for later visual acceptance.

Non-goals:
- No business logic changes.
- No DAO/API changes.
- No printing changes.
- No permission changes.
- No Enter/editable-grid navigation changes.
