# Phase 391 — Item Delete BOM Usage Resolver

## الهدف

عند محاولة حذف مادة بقيت مستخدمة في وصفات تصنيع، يجب ألا يرى المستخدم رسائل تقنية مثل `bom_line=2`. يجب أن يعرف اسم الوصفة/المنتج الذي يمنع الحذف وكيف يحل المشكلة.

## التعديل

- إضافة Resolver داخل `DatabaseConnection` يعيد:
  - الوصفات التي تستخدم المادة كمكوّن.
  - الوصفات التي تكون فيها المادة المنتج النهائي.
- رسالة حذف المادة أصبحت تعرض أسماء الوصفات/المنتجات وأرقام BOM بدل أسماء الجداول الخام.
- إضافة مسار إرشادي واضح: افتح التصنيع > الوصفات ثم احذف سطر المادة أو احذف الوصفة.
- إضافة `bom_usage()` إلى Product Gateway و`item_bom_usage()` إلى ProductService حتى لا تضطر واجهات المستخدم إلى قراءة SQL مباشرة.
- `blocking_total` أصبح يجمع القيم الرقمية فقط حتى لا تتأثر الحسبة بقوائم تفاصيل BOM.

## النتيجة

إذا كانت الفواتير محذوفة وأوامر الإنتاج ملغاة، لكن المادة ما زالت داخل وصفة تصنيع، تظهر رسالة مثل:

> لا يمكن حذف المادة لأنها مرتبطة بعمليات نشطة: وصفات تصنيع تستخدم هذه المادة كمكوّن: #12 منتج تجريبي (عدد أسطر المادة: 2)، افتح التصنيع > الوصفات ثم احذف سطر المادة من الوصفة أو احذف الوصفة نفسها إذا لم تعد مطلوبة.
