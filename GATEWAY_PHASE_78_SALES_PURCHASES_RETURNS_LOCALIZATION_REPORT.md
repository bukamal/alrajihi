# GATEWAY PHASE 78 — Sales / Purchases / Returns Localization

## النطاق
- فواتير البيع.
- فواتير الشراء.
- نافذة الفاتورة.
- مرتجعات المبيعات.
- مرتجعات المشتريات.

## ما تم
- إضافة مفاتيح ترجمة عربية/ألمانية/إنجليزية للمبيعات والمشتريات والمرتجعات.
- تحويل عناوين وأزرار وفلاتر ورسائل الحذف والتنبيهات الأساسية إلى `translate(...)`.
- تعريب/ترجمة رؤوس جداول الفواتير والمرتجعات وبنود الفاتورة.
- الحفاظ على العربية كلغة مصدر وافتراضية، والألمانية ثانية، والإنجليزية ثالثة.

## قيود مقصودة
- لم يتم تغيير منطق العمل المحاسبي للمرتجعات في هذه المرحلة.
- لم يتم إدخال EventFilter أو تعديل Runtime.
- الترجمة التفصيلية للطباعة والتقارير ستبقى لمرحلة لاحقة.
