# Phase 395 — Table Language Direction

## الهدف
تصحيح اتجاه جميع الجداول عند تبديل لغة الواجهة. العربية تبقى RTL، بينما الألمانية والإنجليزية والفرنسية تصبح LTR.

## نطاق الإصلاح
- جداول العرض: `CustomTableView` و `SmartTableView`.
- الجداول التحريرية: `EditableSmartGrid` و `TransactionLineGrid`.
- جداول `QTableWidget` العامة داخل الصفحات والحوارات.
- جدول فاتورة واجهة المطعم البسيطة.
- تحديث الاتجاه أثناء تبديل اللغة من الإعدادات بدون الحاجة لإعادة تشغيل البرنامج.

## القاعدة
لا يوجد جدول يفرض `Qt.RightToLeft` مباشرة. كل الجداول تمر عبر `ui.table_direction_policy`.
