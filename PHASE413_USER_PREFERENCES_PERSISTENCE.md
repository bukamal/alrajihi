# Phase 413 — User Preferences Persistence

## الهدف

تثبيت طبقة حفظ تفضيلات المستخدم التي يجب أن تبقى بعد إغلاق البرنامج وفتحه مجددًا، بدءًا من لوحة التحكم.

## ما تم تنفيذه

- إضافة خدمة مركزية: `core.services.user_preferences_service`.
- تخزين التفضيلات عبر `QSettings` لأنها تفضيلات مستخدم/جهاز وليست إعدادات محاسبية عامة.
- عزل المفاتيح حسب المستخدم الحالي، مع دعم العزل حسب الفرع عند الحاجة.
- حفظ حالة إخفاء أرصدة لوحة التحكم فور الضغط على زر العين.
- استعادة حالة إخفاء الأرصدة عند إنشاء لوحة التحكم.
- حفظ نمط حركة الصندوق في لوحة التحكم: حركة اليوم / الحركة العامة.
- استعادة نمط الحركة عند فتح البرنامج مرة أخرى.

## الحدود

هذه المرحلة لا تغيّر منطق المحاسبة ولا أرصدة الصناديق. التعديل يخص ذاكرة واجهة المستخدم فقط. حفظ أعمدة الجداول وأحجام الـ inline مؤجل للمرحلتين التاليتين حتى يبقى كل نطاق قابلًا للاختبار.
