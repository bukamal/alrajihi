# System-3.2 Purchase Returns Management

تمت إضافة مرتجعات المشتريات المرتبطة بفواتير الشراء الأصلية مع أثر مخزني، مستودعي، مالي، وAudit.

- منع إرجاع أكثر من الكمية المشتراة المتبقية.
- منع الإرجاع إذا الرصيد غير كافٍ في المستودع.
- تسجيل حركة مخزون purchase_return.
- تسجيل حركة مستودع purchase_return_out.
- دعم تخفيض رصيد المورد أو استرداد نقدي/بنكي.
- دعم إلغاء المرتجع بعكس الآثار.
